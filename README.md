# Codificacion-Huffman

Programa realizado con el lenguaje Python sobre la codificación de Huffman el cual es un algoritmo para la compresión de los datos, este algoritmo crea un arbol binario
de nodos, un nodo contiene un cada caracter distinto del texto.
